# IoT Based TDS Water Quality Monitoring System

This project implements a real-time TDS (Total Dissolved Solids) monitoring system using ESP32 and IoT technologies.

## Features
- ESP32 based TDS monitoring
- MQTT communication
- Python MQTT subscriber
- MySQL database logging
- Real-time water quality monitoring

## Technologies Used
- ESP32
- Arduino IDE
- MQTT
- Python
- MySQL

## Files
- tds.ino → ESP32 Arduino code
- tds.py → Python MQTT logger
- TDS Report.pdf → Project report/documentation
